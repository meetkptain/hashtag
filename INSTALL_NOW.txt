OPTION A - SQLite (PLUS SIMPLE) :
─────────────────────────────────
New-Item -Path "database" -Name "database.sqlite" -ItemType "file"

Puis éditer .env :
DB_CONNECTION=sqlite
# Commenter toutes les autres lignes DB_*

OPTION B - MySQL :
─────────────────────────────────
1. Ouvrir MySQL :
   mysql -u root -p

2. Créer la base :
   CREATE DATABASE hashmytag CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
   EXIT;

3. Configurer .env :
   DB_CONNECTION=mysql
   DB_HOST=127.0.0.1
   DB_PORT=3306
   DB_DATABASE=hashmytag
   DB_USERNAME=root
   DB_PASSWORD=votre_mot_de_passe
